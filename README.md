# Software Wrighter Lab - Development

Development and preview repo for the Software Wrighter Lab blog.

## Repository Architecture

| Repo | Purpose | URL |
|------|---------|-----|
| **sw-lab** (this repo) | Development & Preview | `github.com/software-wrighter-lab/sw-lab` |
| **blog-planning** | Drafts & Planning (private) | `github.com/software-wrighter-lab/blog-planning` |
| **sw-lab.github.io** | Publishing (public) | `github.com/software-wrighter-lab/sw-lab.github.io` |

## Local Development

```bash
bundle install
bundle exec jekyll serve --livereload --port 5907
```

Site available at http://localhost:5907

## Publishing Workflow

1. Develop and preview in this repo (sw-lab)
2. When ready to publish, copy to sw-lab.github.io and push
3. GitHub Pages builds and deploys automatically
