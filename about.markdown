# Software Wrighter Lab

A blog exploring AI coding agents, systems programming, and practical machine learning.

## About the Author

Mike Wright is a software engineer with a passion for exploring the intersection of AI and software development. This blog documents experiments, insights, and practical applications in the rapidly evolving landscape of AI-assisted programming.

## Topics Covered

- **AI Coding Agents**: Exploring how AI assistants can help with software development
- **Systems Programming**: Low-level programming, performance optimization, and computer architecture
- **Machine Learning**: Practical applications and experiments with ML models
- **Throwback Thursday (TBT)**: Revisiting classic programming languages and techniques
